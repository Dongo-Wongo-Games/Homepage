# Current Project
## Tabernacle
<p align="center">
  <img src="images/Witch_TP_Angled.png" alt="Tabernacle Witch" width="1000">
</p>

**Tabernacle** is a cooperative horror game where players must explore dangerous environments, complete objectives, evade deadly entities, and survive together.
### Features
- Online cooperative multiplayer
- Proximity voice communication
- Multiple unique monster types
- Dynamic AI behaviors
- Atmospheric horror environments
